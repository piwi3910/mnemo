Architecture Overview

This document describes the system architecture for the Mnemo project.

Components

The frontend uses React with CodeMirror for editing.

Summary

  • Default: Sidebar | Preview | Graph + Outline
  • Edit mode: Sidebar | Editor | Preview (graph + outline hide)
  • Edit button in the preview toolbar, "Done" button to exit edit mode
  • Graph shows local view by default with toggle to full graph
  • Clicking a graph node navigates to that note + highlights it
  • Backlinks/outgoing links stay below the main content area
